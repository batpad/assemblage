Help

Starting the App

Click on the ‘build’ link in the header to start the interactive platform


Viewing the database of objects

The database contains all the objects, fragments and assemblages. These are stored in the online repository and occupy the left column on the screen titled ‘database’.

Note: Users can create and save new assemblages. If you wish to upload new objects and fragments, email Ranjit at ranjit@buildbrowse.com.

Filtering and Search

You can use the filtering and search features to modify the list of objects.

The dropdown menu for the filtering list contains the following options

- All
- Objects: Articles or found items (in this case accessed from a market), salvaged from a particular market.
- Fragments: A part of an object, usually incomplete, usually further devaluing the object.
- Assemblages: The interconnection of two or more objects/fragments which results in the user's new artistic composition.
- My Objects: The list of exported assemblages (PNG images) that a user has created

Advanced Search

Clicking the ‘More’ button to the right of the search box opens an optional advanced search where you can narrow down your search.

List of Object Thumbnails

Each object within the list displays a representative thumbnail, name, width and a ‘View More’ icon (represented by an +) that appears on mouseover or hover). Clicking the ‘View More’ icon takes you to a new browser tab, where you can view detailed information about the object.


Using the Canvas Editor

The canvas editor is the virtual editor where you can drag objects and create and save new imaginations.

Dragging objects:
If your mouse if hovered over the thumbnail of an object, your cursor will display a drag symbol, indicating that you can drag the object to the canvas editor on the right. Alternately, you can double click on the thumbnail and it will load the corresponding image in the canvas.

You can drag as many objects as you like to the canvas, one at a time.

Tools and Editing Features:

The toolbar on the left enables you to comfortably edit your objects. Hovering over each icon shows its functionality.

6Undo: Allows you to undo your actions multiple times.

Keyboard shortcut: z

5Redo: Allows you to redo the actions that you previously undid.

Keyboard shortcut: y

*Join: To join two or more objects, you first need to select them (either by drawing a ‘box’ around the objects you wish to join or keeping the Shift key pressed whilst clicking the objects you wish to join) and then click the Join icon.

Note: Once you join objects, the only way to unjoin them is by clicking Undo.

Keyboard shortcut: j

Bring to front: Select an object with your cursor and click bring to front if you want it to appear over an overlapping object.

Keyboard shortcut: f

Send to back: Select an object with your cursor and click send to back if you want it to appear under an overlapping object.

Keyboard shortcut: b

(Scale-up: This mimics the functionality of the scale slider on the bottom right of the canvas editor, i.e. it increases the size of the canvas.

Keyboard shortcut: q

'Scale-down: This mimics the functionality of the scale slider on the bottom right of the canvas editor, i.e. it decreases the size of the canvas.

Keyboard shortcut: w

3Delete: Select an object with your cursor and click delete to remove it from the canvas. You can press undo to cancel this action.

Keyboard shortcut: delete

Pan: Clicking on pan will change the cursor to display a pan symbol and highlights the pan icon in the toolbar. Being in pan mode allows you to drag your cursor and access hidden parts of the canvas, if you’re zoomed in.

Note: When in pan mode, clicking on objects does not select them as usual. To exit pan mode, press the Escape key or click the Pan icon once more.

Keyboard shortcut: p to activate and escape to deactivate

"New: Creates a new canvas/project in the canvas editor

Keyboard shortcut: alt + n

2Open: Opens a dialog with a list of your past saved projects

Keyboard shortcut: alt + o

#Save: Saves the current project you’re working on

Keyboard shortcut: alt + s

$Save as: Saves a copy of the current project you’re working on

Keyboard shortcut: alt + s

4Export: Exports your work. This gets added to the database as a new assemblage, in a PNG image file format. Do note that this is different from saving a project, whose different objects can be modified. When are you done with your project and want to add it to the database for others to access, you can hit export.

Keyboard shortcut: alt + e

.Print: Opens a new browser tab where you can take a printout of the current state of your work

Keyboard shortcut: alt + r

Zooming: You can use the slider titled ‘Scale’ on the bottom right of the canvas editor to increase or decrease the canvas dimensions, which will proportionally modify the size of the objects.

Scrollbars: In addition to pan, the scrollbars are useful to navigate within the canvas editor. Horizontal and vertical scrollbars are provided on zoom.

Customising your canvas editor: The dropdown menu next to the Print button contains a list of different swatches, that you can use to customise the background of the canvas editor.
